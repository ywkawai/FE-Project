\section{Available test cases}
\label{sec:available_test_cases}
%------------------------------------------------------

Several test cases and sample experiments are prepared under
\verb|model/atm_nonhydro3d/test/case/|.
These cases are useful not only for checking the installation and basic
operation of \scaledg, but also as templates for user-defined experiments.
Users are encouraged to copy one of these directories and modify the
configuration files, initial conditions, physical processes, grid settings,
and output settings according to their own purposes.

The prepared test cases cover regional and global configurations, dynamical-core
test problems, tracer advection tests, turbulence experiments, and idealized
planetary-atmosphere experiments.
Examples are listed in Table \ref{tab:available_test_cases}.

A typical procedure is to move to a target directory and execute

\begin{verbatim}
$ make
$ make run
\end{verbatim}

Some test cases also provide visualization scripts or additional sample
configuration files. For example, if a visualization target is prepared in
the Makefile, figures can be generated by

\begin{verbatim}
$ make vis
\end{verbatim}

The settings in these test cases are not intended to cover all possible
scientific applications. They should be regarded as starting points for
constructing new experiments.
